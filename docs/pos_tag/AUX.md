# Trợ động từ 

Auxiliary

Nhãn UD: AUX  

Gồm nhiều loại:

* Tense auxilaries: has (done), is (doing), will (do)
* Passive auxiliaries: was (done), got (done)
* Modal auxiliaries: should (do), must (do)
* Verbal copulas: He **is** a teacher
