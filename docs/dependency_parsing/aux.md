# aux

* Ký hiệu UD: aux

Quan hệ giữa trợ động từ với động từ

``` 
                __aux__
              ↙         \
Câu_chuyện có_thể     theo bốn khả_năng.
```

**Từ phiên bản v2:** Các trợ động từ trong câu bị động cũng được gán nhãn `aux`, mặc dù khuyến nghị nên sử dụng nhãn con
`aux:pass`
